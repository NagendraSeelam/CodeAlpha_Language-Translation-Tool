# Project Report

## Project Title
Language Translation Tool

## Objective
To develop a GUI based application capable of translating text between different languages.

## Technologies Used
- Python
- Tkinter
- Google Translate API

## Methodology
1. User enters text.
2. User selects source language.
3. User selects target language.
4. Application translates text.
5. Result is displayed.

## Conclusion
The project successfully translates text between multiple languages.
